# @social 🚀

**Enterprise Social Media Management Platform** — Manage all your social media accounts from one powerful command center.

## ✨ Features

- **12+ Platform Support** — Instagram, X/Twitter, Facebook, LinkedIn, TikTok, YouTube, Pinterest, Snapchat, Threads, Reddit, Discord, Telegram
- **Smart Scheduler** — AI-powered optimal posting times with visual calendar
- **Deep Analytics** — Real-time performance metrics, growth trends, and competitor benchmarking
- **Unified Inbox** — All comments, DMs, and mentions in one stream
- **AI Content Writer** — Generate platform-optimized captions and hashtags
- **Team Collaboration** — Role-based access and approval workflows
- **White-Label Reports** — Client-ready branded PDF reports

## 🎨 Design

Premium dark UI with:
- Glassmorphism panels and animated particle backgrounds
- Electric indigo + cyan color system
- Smooth micro-animations and hover effects
- Fully responsive layout
